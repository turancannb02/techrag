# TechRAG Demo Corpus

## 3GPP TR 38.901 (Channel Model)

TR 38.901 studies channel modeling for frequencies from 0.5 to 100 GHz.
It covers pathloss models, LOS probability, shadow fading, O2I penetration,
clustered delay line assumptions, and antenna array modeling guidance.

## 3GPP TS 38.300 (NR and NG-RAN)

TS 38.300 provides a high-level Stage-2 overview of NR and NG-RAN architecture,
including functional split, interface roles, and major system procedures.

## Retrieval-Augmented Generation

RAG systems retrieve relevant context from a corpus before generation.
Hybrid retrieval commonly combines dense semantic retrieval and sparse keyword
matching (BM25) to improve robustness.
